The Crimson Key is a Biome Key that the player can obtain from any enemies that spawn in the Crimson biome. This key will unlock a Crimson Chest in the Dungeon, after Plantera has been defeated. Before defeating Plantera in a world, like all other biome keys, the tooltip will say "It has been cursed by a powerful Jungle creature", and is unable to be used.
